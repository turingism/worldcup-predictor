# ⚽ 比赛日运行手册

<p align="right"><a href="./RUNBOOK.md">English</a> · <strong>简体中文</strong></p>

> 比赛日跑预测器的一页纸操作指南。项目是什么看主 [README](../README.zh-CN.md)。

### ▶️ 启动
```bash
cd ~/worldcup-predictor && python3 app.py        # 开 http://127.0.0.1:8000
```
- 端口固定 **8000**（别用 5000，被 macOS AirPlay 占）。
- 页面空白/卡住？ **`Cmd+Shift+R` 硬刷新**（清前端缓存）。

### 🤖 开着网页就自动发生（无需动手）
| 频率 | 干什么 |
|---|---|
| 每 60 秒 | 看板刷新实时比分；LIVE 卡的 in-play 胜平负随比分跳 |
| 每 30 分钟 | 后台抓一次 ESPN 盘口，积累开盘/闭盘线（喂 CLV）|
| 有新完赛 | 自动重训模型 → 后台重算夺冠 90% 区间 |

### 🖱️ 按钮 / Tab 速查
- **📋 赛事看板**（首屏）：🔴 正在比赛 / 🟡 即将开赛 / ✅ 已结束。每场「看预测」弹比分矩阵。
- **🔄 刷新事实数据**：手动秒拉 ESPN 完场赛果（有新的就重训）。
- **🏆 夺冠概率**：蒙特卡洛点估 + 贝叶斯 90% 区间（whisker 图，「↻ 重新计算」手动刷）。
- **🎯 预测验证**：赛前冻结预测 vs 实际，逐场打分（开赛初期看 RPS/校准，别盯命中率）。
- **💹 市场对标**：模型 vs 闭盘线 + CLV；价值/Kelly 面板 **默认诚实锁定**。

### 🔧 两个开关（环境变量）
```bash
READONLY=1 python3 app.py        # 只读分享模式：禁所有写接口（对外分享用）
MARKET_UNLOCK=1 python3 app.py   # 强制打开价值/Kelly 面板（仅演示，大红警示，非建议）
```

### 🩹 排错速查
- **看板卡在"加载…"** → 硬刷新；还不行就重启 `app.py`。
- **市场 tab 慢/空** → 开赛初期正常（还没有"已完赛且抓到盘口"的场次 → "样本不足"）。
- **刷新报 403** → 你在 `READONLY=1` 模式（写接口被刻意禁用）。
- **数据陈旧** → 点「🔄 刷新事实数据」，或重启 `app.py`。
