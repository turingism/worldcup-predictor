## 2026-06-21 — 📈 "命中率为何提升"归因 + 5 杠杆优化 sprint（多 agent，全部用数字否决）

用户观察到赛果命中率从开赛初期 37.5%(n=8) 升到 55.6%(n=36)，问为什么、并要继续优化算法。

### 根因：不是优化，是向真值回归（引擎自 6-12 起一行未动）
- **统计定论**：37.5% 的 95%CI [13.7, 69.4] 与 55.6% 的 95%CI [39.6, 70.5] **都包含长期真值 59.7%**——两个数在统计上是同一个数，"提升"=样本从 8→36 的均值回归，纯大数定律。
- 样本外回测复现真值：argmax 命中 **59.50%**（n=2420），与文档 59.7% 吻合。现役 55.6% 仍略低于真值，后续会继续爬。
- **结构分解**：36 场里 11 场平局(30.6%)全部结构性丢失（argmax 几乎不输出"平"）；**非平局 25 场命中 20=80%**。损失集中在平局，模型判断力正常。

### 5 杠杆优化 sprint（每个 agent 独立跑 as_of 防泄漏回测 + 对抗验证；全部 reject）
基线 cutoffs=[2024-03,2024-11,2025-08] horizon=270 hl=730，池化 n=2420 RPS=0.16371 Acc=0.595。采纳门槛=池化 RPS 改善>0.0008 且最近 cutoff(2025-08) 不变差。

1. **平局决策规则族**（δ加成 / 双条件 / 仅硬币局判平）→ **REJECT**。决策不改概率故 RPS 恒等 0.16371；最优 coin τ=0.07 在最近 cutoff 改 61/736 决策但胜负相抵净增 0 Acc，且增益随留出轮换剧烈漂移(+0.0107/+0.0031/+0.0000)=样本特异过拟合。**argmax 已是命中率最优。**
2. **中立场名义主场优势校准** → **REJECT（但发现真实偏差）**。neutral=True 子集(n=743) 主队槽位实际胜率被低估 **+2.24pp**（pred 38.1% vs act 40.4%），方向真实；但最优 δ=0.10 tilt 只换池化 RPS **+0.0002**（门槛 1/4），且是窄峰(δ≥0.25 转负)。**真实但经济意义太小，不落地。**
3. **Dixon-Coles ρ 近期性重拟** → **REJECT**。最优 rho_hl=180 改善仅 **+0.000008**（门槛 1/100），最近 cutoff RPS 平 Acc 不变；放更长 hl=1460 反更差。4 格低分校准无可利用偏差（低平局质量仅欠 0.5pp，二项噪声内）。**惰性杠杆，印证 half_life 730-1460 平底盆地。**
4. **本届 36 场校准审计**（诊断）→ **REJECT 任何重校准**。表观 ECE(5桶)=4.47% 看似高于历史 1.06%，但蒙特卡洛零分布证明：**完美校准模型在 n=36 下 ECE 均值就有 6.02%、95 分位 10.72%**，观测 4.47% 反而**低于**零分布均值(p=0.72)；RPS 0.172 也低于自洽零分布 0.180(p=0.60)。**本届概率完全校准，44.6%/0.172 是结构性平局盲区+小样本，非失准。** 与已否决的 isotonic 同向。
5. **分洲/分赛事差异化 half_life** → **REJECT**。最优 A1_confed 池化 RPS 改善仅 -0.00001（门槛 1/80，噪声），2024-11 反变差；B1_tier 反而 +0.00038。给衰减加自由度=平底上抬方差无信息增益，与 comp_weights 否决同理。

### 最终定论
命中率与 RPS 已贴近本数据下双泊松模型的信息上限；可解释误差几乎全是结构性平局盲区+小样本噪声，**无任何可校准/可调参修复的系统偏置**。引擎保持现状，5 杠杆诚实 reject 是正确且预期的结果。

## 2026-06-21 — 🔮 玄学占卜「七法忠实排盘」重写 + 卡片对齐 + 文档双语更新

延续玄学擂台的**诚实定位（B 类）**：优化目标是**对传统术数排盘的忠实度（fidelity）**，
**不是**提高命中率（提高命中率＝灌真实球队先验＝把玄学变成伪装基线，违背诚实定位）。
三条红线全守：① 完全确定性可复现（禁随机）② 绝不读赛果 ③ 不灌实力先验（队名仅 hash 进取用层，分主客 + 区分同时刻不同对阵）。

### 七法引擎全部重写为忠实排盘（`xuanxue.py`，每法带独立前缀表互不冲突；`ganzhi.py` 零改动）
- **梅花易数**：时间起卦三步（年支+月+日 %8 上卦／+时 %8 下卦／总和 %6 动爻）+ 动爻定体用 + 互卦 + 变卦 + 卦气旺衰。
- **射覆**：梅花路线占时起卦，由卦五行**生成数 × 旺衰 × 动爻**"射"出进球数（覆物之数）。
- **周易·六十四卦**：以数起卦得本卦 → 动爻 → 变卦；综合 64 卦卦义吉凶 + 动爻爻位中正 + 体用生克 + 卦变升降。
- **六爻纳甲**：京房八宫（程序生成 64 卦）定世应 + 纳甲配五行六亲 + 六兽 + 月建日辰旺衰生克 + 旬空/月破/冲合/暗动，世(主)/应(客)取用。
- **奇门遁甲**：定阴阳遁/三元/局数 → 地盘三奇六仪 → 天盘九星/八门/八神 → 值符值使旬首遁仪 → 主(日干)/客(时干)落宫断旺衰 + 格局（青龙返首/飞鸟跌穴/三奇得使/入墓/门迫/伏吟/五不遇时）。
- **大六壬**：月将加占时排天地盘 → 起四课 → **九宗门**发三传 → 昼夜贵人顺逆布十二天将 → 日干(主)/日支(客)取用断课体。
- **紫微斗数**：节气月+时辰定命宫/身宫（五虎遁定干）→ 纳音起五行局 → 安紫微/紫微系/天府系 14 主星 → 庙旺利陷 + 生年四化落宫，命(主)/迁(客)取用。农历以节气月/干支日序近似（已标注）。
- **修边界 bug**：奇门值符遁仪落中 5 宫时 `_QM_GATE_HOME[5]` KeyError（历史回测触发）→ 中 5 无门寄坤 2。

### 诚实铁证保持（最关键）
- 历史 **49438 场**回测，七法胜负命中 **31.6%–38.3%**（贴随机 33.3%），**全部低于无脑基线「永远押主胜」49.0%**；
  精确比分 1.9%–6.1% 低于「全押最常见比分 1-0」10.3%。**忠实度大升、却照样赢不了基线——正是诚实定位要的结果。**

  加 `.xxmt{table-layout:fixed}` + `<colgroup>` 固定列宽（待测 50/25/25、已完赛 40/20/22/18），12 张卡三列完全对齐。

### 文档（双语）

- **解读层 Elo 口径修正**：`insights._elo_rank` 原在全 ~250 队里排 Elo，与我方"48 队内夺冠排名"不同口径（apples-to-oranges）。改为**只在 48 支参赛队内部排 Elo**，偏差数才真正可比（挪威 +11、法国 −7 均为 48 队内）。
